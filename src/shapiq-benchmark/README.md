# shapiq-benchmark

A benchmark suite for evaluating the performance of model-agnostic estimators for game-theoretic concepts such as the Shapley value, Shapley interactions, Banzhaf value, and others.
The benchmark is build on top of [`shapiq`](https://github.com/mmschlk/shapiq).

## 🛠️ Install
Clone the repository and install the dependencies:
```bash
uv sync
```
