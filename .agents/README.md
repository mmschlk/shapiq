# On the use of agentic coding in shapiq

We use agentic tools for the development of shapiq.
To make the codebase maintainable and understandable, we ensure that all concepts and requirements are specified in-detail.
To this end, we use (some of) the great [skills](https://github.com/mattpocock/skills) developed by Matt Pocock.

All APIs and algorithms are designed, specified and reviewed by the core maintainers, AI-driven workflows are only used to an extent which still permits full code ownership.
