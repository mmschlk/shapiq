# Baseline Scores And Centered Attributions

Explanations carry the empty-coalition value as a dedicated ``baseline``, and every attribution is defined on the centered game ``v - v(empty)``, following the game-theoretic convention that the empty coalition has value zero. Before this decision the order-0 attribution slot conflated three different things: the baseline itself (SV, STII, k-SII, kADD-SHAP, JointSV, and the Moebius transform stored ``v(empty)`` there), a genuine index-defined constant (FBII's fitted intercept, offset by ``v(empty)``), and the grand-coalition value (Co-Moebius). Game theory treats the baseline as bookkeeping about the game, not as an attribution of any interaction — every shipped index is invariant to centering on nonempty interactions, since discrete derivatives and bloc marginals annihilate constants and the regressions fit centered responses — so the baseline now travels on the ``ExplanationArray`` (populated by every explainer from its seed evaluations), and the order-0 slot means one thing: the index's own attribution to the empty interaction on the centered game.

Only two shipped indices have one. FBII's order-0 attribution is the intercept of its centered unconstrained fit — previously reported with ``v(empty)`` added back, now reported pure, with the baseline alongside. The Co-Moebius transform's is the grand total ``v(N) - v(empty)``, computed by its own anchor kernel on the centered game. Everything else stopped carrying an order-0 attribution: the Moebius transform's empty mass is identically zero on the centered game (reconstruction reads ``v(T) = baseline + sum of masses over nonempty subsets``), and the efficient indices' totals now read cleanly as the sum over all attributions equalling ``v(N) - v(empty)`` with the baseline stated separately. ``explanation(())`` returns the genuine order-0 attribution where one exists and raises otherwise; ``explanation.baseline`` is always populated by shapiq's explainers and optional on hand-built arrays.

A welcome consequence: the order-0 asymmetry that qualified extensional index equality (ADR 0009) largely dissolves. ``SII(order=1)`` and ``SV()`` explanations now have identical coverage — both attribute to singletons and both carry the baseline — so the equality of the objects matches the observable shape of their explanations, with FBII's intercept remaining the one honest per-index order-0 difference.
