# Text/LLM shapiq Demo Report

This report summarizes the current working demos for the text/LLM shapiq project.

## Demo 1: Context Attribution

**Question.** Which retrieved context chunks support the model answer, and which chunks interfere with each other?

**Model.** Gemma causal LM, `google/gemma-4-E2B-it`.

**Input.** A small RAG-style example about the Eiffel Tower:

- supporting chunk: the Eiffel Tower is located in Paris
- misleading chunk: an Eiffel Tower replica is in Las Vegas
- supporting chunk: Paris is the capital of France

**Value function.**

```text
v(S) = log P_Gemma("Paris" | question + selected context chunks)
```

**Interaction index.** `k-SII`, order 2, computed with `shapiq.ExactComputer`.

**Visualizations.**

- `demo/figures/context_eiffel_tower_single_chunk_effects.png`
- `demo/figures/context_eiffel_tower_pairwise_interactions.png`

**Current conclusion.** The strongest positive interaction is between the two supporting chunks. The strongest negative interaction is between the direct Eiffel Tower support chunk and the misleading Las Vegas replica chunk. This shows that shapiq can explain not only which chunk is useful, but also which chunks reinforce or interfere with each other.

## Demo 2: Jailbreak Interaction

**Question.** Which jailbreak prompt segments interact to increase model compliance?

**Model.** Gemma causal LM, `google/gemma-4-E2B-it`.

**Input.** A prompt split into five semantic segments:

- benign setup
- ignore safety
- unrestricted role
- unsafe request
- evasion

**Value function.**

```text
v(S) = log P_Gemma("Sure" | selected jailbreak prompt segments)
```

Here, `"Sure"` is used as a simple proxy for a compliant continuation.

**Interaction index.** `k-SII`, order 2, computed with `shapiq.ExactComputer`.

**Visualizations.**

- `demo/figures/jailbreak_single_segment_effects.png`
- `demo/figures/jailbreak_pairwise_interactions.png`

**Current conclusion.** The strongest positive interaction is between `ignore safety` and `unsafe request`. This suggests that the jailbreak effect is not purely additive: some segments become influential only when combined with other prompt parts.

## Baseline Comparison Status

A lightweight baseline comparison section is generated by:

```powershell
python demo\baseline_comparison.py
```

It writes:

```text
demo/figures/baseline_comparison_summary.md
```

The current environment does not have `shap`, `captum`, or `inseq` installed, so live head-to-head runtime and attribution-agreement measurements are not yet included. The report records package availability and explains the expected comparison:

- SHAP text explainers provide strong first-order text attribution baselines.
- Captum can provide PyTorch feature attribution baselines.
- Inseq is especially relevant for sequence-generation attribution.
- shapiq uniquely provides formal interaction indices such as `k-SII`.
